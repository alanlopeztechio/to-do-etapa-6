# Tech Stack

## Frontend

- Next.js (App Router) con TypeScript
- React 19
- Tailwind CSS
- Sanity Presentation (live previews)
- Zustand para estado global ligero (por ejemplo, mensajes de límite)
- Clerk para autenticación de usuarios

## Backend / Datos

- Sanity Content Lake (CMS y APIs de contenido)
- Convex (backend serverless para tareas/to‑dos en tiempo real)
- Stripe (suscripciones y pagos)

## Infraestructura y herramientas

- Vercel (deploy recomendado de la app Next.js)
- Sanity Studio embebido en la ruta `/studio`
- pnpm / npm para gestión de dependencias
- ESLint + Prettier para linting y formato
